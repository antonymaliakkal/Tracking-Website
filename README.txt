STEP 1 : Open the folder "tracker" in an IDE(VS Code ,Replit etc...)
STEP 2 : install all the required packages mentioned in "req.txt" either by pip install or by typing "pip install -r req.txt" in the terminal
STEP 3 : Run the main.py file
STEP 4 : The website will be running in the address "http://127.0.0.1:5000/"
STEP 5 : Search the link in some browser(Chrome,Edge etc..)
STEP 6 : This link will tak you to the login/sign-up page.
